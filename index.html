How It Works

1

Publish

Push manifests and prompt files to IPFS. Immutable CIDs establish the authorized set of agent instructions.

2

Propose

Each manifest is one governance proposal in your SubDAO. Vote, queue, and execute through a Timelock.

3

Discover

Agents, the CLI, and web app read via subgraph first and fall back to RPC for a consistent view.

4

Use

MCP tools return content by CID so agents can orchestrate tasks with the community-approved version.

Get Started

Connect & Publish

# Connect your wallet
sage wallet connect --type cast
sage doctor

# Publish a library
sage library push manifest.json --pin

Run MCP Server

# Start HTTP MCP server for AI agents
sage mcp start


# Or run stdio server for Claude Desktop
node packages/cli/src/mcp-server-stdio.js
